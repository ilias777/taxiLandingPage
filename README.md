## To try this website

Clone the repo
```shell
git clone https://github.com/ilias777/taxiLandingPage.git
```

Navigate to the folder
```shell
cd taxiLandingPage
```

Install npm packages
```shell
npm install
```

Run dev server to see the page
```
npm run dev
```
